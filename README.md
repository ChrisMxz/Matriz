## Slot de memoria

**Definidos como:**

 - `m1` Es un slot 1
 - `m2` Es un slot 2
 - `m3` Es un slot 3

 - `ans` : *Este almacena la respuesta de ultima operación realizada*

## **Operaciones  **

Suma de matrices 
`m1 + m2`

Resta de matrices
 `m1 - m2` 

Producto Punto
`m1 * m2` 

Producto Cruz
`m1 x m2`
